# Airfoil AI Platform — Frontend

This is the React frontend for the Airfoil AI Platform.

For full setup instructions, features, tech stack, and API documentation, see the [root README](../README.md).

## Quick Start

    npm install
    npm start

Frontend opens at http://localhost:3000

Requires the backend running at http://localhost:8000. See root README for backend setup.

## Tech Stack

- React 18
- Three.js + React Three Fiber (3D airfoil viewer)
- Recharts (drag polar and Reynolds sweep charts)
- Framer Motion (tab animations)
- Axios (API calls)
